# @fluxure/landing

Marketing landing page for Fluxure. Built with SvelteKit and adapter-static.

## Pages

- `/` — Landing page
- `/privacy` — Privacy policy
- `/terms` — Terms of service

## Development

```bash
pnpm dev      # Vite dev server
pnpm build    # Static build to build/
pnpm preview  # Preview production build
```

## Deployment

Deployed to Cloudflare Pages. See [DEPLOY.md](../../DEPLOY.md) for details.
