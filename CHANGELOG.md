# Changelog

All notable changes to Fluxure are documented in this file.

Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-03-20

Initial release.

- Multi-tenant SaaS with JWT auth, Google OAuth, and email/password signup
- Greedy scheduling engine with quality scoring and timezone-aware slot generation
- Natural language quick-add, public booking pages, and scheduling templates
- Google Calendar sync (push notifications + polling fallback)
- PWA with offline caching, real-time WebSocket updates
- Free and Pro tiers with Stripe billing and 14-day trial
- Prometheus metrics, structured logging, self-hosting support
- GDPR compliance: data export, account deletion, consent tracking
